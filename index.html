App Permissions

For AppLock to function optimally and deliver smooth user experience, it will require a number of device permissions to be granted. Some permissions are only requested based on the functionality to be enabled. The permissions are listed below:

a. Modify or delete the contents of your USB storage

This permission allows users to delete AppLock wallpapers stored in SD card.

b. Take pictures and videos

Used by AppLock to capture intruder selfies as might have been enabled by the user

c. Read the contents of your USB storage

This permission allows users to view intruder selfie photos as captured by the application.

d. Full network access

This permission allows AppLock to access network for achieving relative services e.g. providing version update notification.

e. Run as startup

This permission allows AppLock services to run at startup.

f. Draw over other apps

This permission allows AppLock to display windows over other apps

g. Read, modify and delete contents in the SD card

This permission allows AppLock to read, modify and delete intruder selfies in SD card as might be required by the user.

h. System alert window

Used to send instant notifications to users by drawing over other messages.
